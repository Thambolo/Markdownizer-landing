# Markdownizer Landing Page

This is the landing page for Markdownizer - a privacy-first browser extension that converts any webpage into clean Markdown.

## Live Site

The site is deployed via GitHub Pages at:
**https://thambolo.github.io/Markdownizer-landing/**

## Deployment

The site is automatically deployed to GitHub Pages when changes are pushed to the `main` branch using GitHub Actions.

### Manual Deployment

You can also trigger a manual deployment from the Actions tab in GitHub by running the "Deploy to GitHub Pages" workflow.

## Local Development

Simply open `index.html` in your browser to view the site locally.
